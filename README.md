# Tutorial: Manejo del DOM en JavaScript

Este tutorial interactivo enseña los conceptos fundamentales de la manipulación del Document Object Model (DOM) usando JavaScript.

## Contenido del Tutorial

1. **Selección de Elementos** - Aprende a seleccionar elementos usando `getElementById`, `getElementsByClassName`, `querySelector` y `querySelectorAll`.

2. **Modificación de Contenido** - Cambia el contenido de los elementos usando `textContent` e `innerHTML`.

3. **Modificación de Estilos** - Modifica estilos CSS directamente y maneja clases con `classList`.

4. **Crear y Eliminar Elementos** - Crea nuevos elementos con `createElement` y `appendChild`, y elimínalos con `removeChild`.

5. **Manejo de Eventos** - Responde a eventos del mouse y teclado usando `addEventListener`.

6. **Manipulación de Formularios** - Lee y valida datos de formularios.

## Cómo usar este tutorial

1. Abre el archivo `index.html` en tu navegador web
2. Interactúa con cada sección haciendo clic en los botones
3. Abre la consola del desarrollador (F12) para ver los logs de cada interacción

## Archivos

- `index.html` - Estructura HTML del tutorial
- `styles.css` - Estilos CSS del tutorial
- `script.js` - Código JavaScript con ejemplos de manipulación del DOM

## Tecnologías

- HTML5
- CSS3
- JavaScript (Vanilla)
